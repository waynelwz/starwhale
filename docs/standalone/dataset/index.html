Starwhale Dataset

dataset.yaml Definition​

FieldDescriptionRequiredDefault ValueType
versionstarwhale api version, today only support 1.0❌1.0String
namestarwhale dataset name✅String
data_dirdata directory✅String
data_filerthe filter for data files, support regular expression✅string
label_filerthe filter for label files, support regular expression✅string
processthe class import path which is inherited by starwhale.api.dataset.BuildExecutor class. The format is {module path}:{class name}✅String
descdescription❌""String
attr.batch_sizedata batch size❌50Integer
attr.alignment_sizeevery section data alignment size❌4kString
attr.volume_sizedata volume size❌64MString

Example:

name: mnist

data_dir: data
data_filter: "t10k-image*"
label_filter: "t10k-label*"
process: mnist.process:DataSetProcessExecutor

desc: MNIST data and label test dataset
attr:
  batch_size: 50
  alignment_size: 4k
  volume_size: 2M
